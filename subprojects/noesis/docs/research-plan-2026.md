# Noesis Research Plan 2026

## Objective

Turn Noesis into a serious AGI-era social research program with an observable
economic substrate layer.

## Workstreams

### 1. Symbiosis Theory

- refine human-machine role taxonomy
- expand institutional redesign scenarios
- connect subjectivity and limits questions back into the umbrella

### 2. Six-Factor Observatory

- specify indicators for each factor
- choose real-time vs slice-based data cadence
- build first static dashboard and map prototypes
- prepare the path toward live data ingestion

### 3. Communicative Coordination Economy

- define the model more sharply
- compare it against price-only and planning-only systems
- identify first candidate domains:
  - infrastructure
  - education
  - healthcare
  - public allocation

### 4. Source Discipline

- maintain high-trust source pack
- keep low-trust signals in idea lanes only
- update frontier project landscape quarterly

## First Deliverables

- six-factor static observatory prototype
- current progress synthesis
- open questions registry
- first communicative-coordination memo
- source-pack and open-project landscape docs
