# Noesis Open Questions 2026

## Human-Machine Symbiosis

- Which social functions remain legitimacy-critical even if machine capability
  rises sharply?
- What kinds of institutions become more important rather than less important
  under machine abundance?
- Where does co-agency outperform either automation or human-only control?

## Machine Limits And Subjectivity

- Which practical limits matter more than formal limits in real AGI-era
  systems?
- How should society behave under moral uncertainty about machine subjectivity?
- Which governance structures are robust whether or not machines become
  subject-like?

## Six-Factor AI Economy

- Which factor becomes the dominant bottleneck first: compute, energy, data, or
  talent?
- How should the observatory distinguish direct measurements from inferred
  indices?
- Which regions are structurally advantaged across multiple factors at once?

## Communicative Coordination And New Economic Models

- Which economic domains fail when price is treated as the only adequate
  coordination signal?
- How should AI-assisted deliberation avoid becoming elite-managed theatre?
- Which legitimacy tests should a communicative coordination economy satisfy?

## Strategy

- Which Noesis outputs should stay research-first and which should become
  public-facing tools?
- How much of the observatory should be narrative and how much should be data
  product?
