# Economy

Use this area for the material and economic substrate of AGI-era society.

This is the integrated Stoicheia-derived subproject inside Noesis.

It should cover:

- data
- storage
- compute
- algorithms
- energy
- talent

The point is not generic AI economics commentary.

The point is to understand how the real substrate of machine intelligence shapes
human-machine symbiosis, institutions, and civilizational power.
