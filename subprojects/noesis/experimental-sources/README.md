# Experimental Sources

This directory is for lower-trust but potentially idea-generating sources.

## Strict Rule

Everything in this directory is:

- low trust
- brainstorming only
- not admissible as direct evidence
- not usable as a standalone basis for claims, ranking, or governance decisions

## Mandatory Header

Every note stored here must begin with a warning block equivalent to:

- `TRUST_LEVEL: LOW`
- `USE: BRAINSTORMING ONLY`
- `NOT_FOR: DIRECT EVIDENCE OR DECISION MAKING`
- `REQUIRES: INDEPENDENT HIGH-QUALITY CONFIRMATION`

If a note does not contain that warning, treat it as invalid.
