# Peras - AI Frontier Signal Engine

Open-source scaffold for an AI signal platform that combines frontier academic
research, business/industry movement, historical pattern analysis, and AI
judgment to help identify high-upside directions for builders, founders, and
investors.

Chinese shorthand: `前沿信号引擎`

## Portfolio Position

Peras is now one of the three subprojects inside `Polis`, Duckermind's
umbrella program for intelligence economy and coordination systems.

Inside Polis:

- `Peras` owns the frontier signal and ranking layer
- `Agora` owns the market and delivery layer
- `Noesis` owns the symbiosis and institutional observatory layer

## Stage

Stage 1: project charter, source architecture, and MVP intelligence-platform framing.

## Core Question

Can we build a disciplined platform that continuously ingests high-quality,
real-time frontier signals and produces better judgments about which AI
directions, products, and markets have the highest future value density?

## Goals

- aggregate frontier academic, commercial, and industry signals into one system
- combine live data with historical pattern analysis instead of treating each new cycle as unprecedented
- use AI synthesis and forecasting to rank directions by future value, not just present hype
- produce periodic, traceable judgment updates for founders and investors

## Direction

Peras should not become a news feed or a copycat trend dashboard.

It should become a judgment engine with:

- source quality discipline
- explicit uncertainty
- periodic updates
- thesis cards
- evidence-backed ranking logic
- a computability / complexity layer that keeps future-value ranking grounded in
  real machine constraints rather than pure hype
