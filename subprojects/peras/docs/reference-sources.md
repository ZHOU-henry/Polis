Peras should be built on source classes, not on one-off links.

## Usage Rule

Peras should prioritize:

- official APIs
- official documents
- high-signal primary sources
- recurring sources that can survive long-term productization

It should avoid becoming dependent on:

- social-media noise
- unsourced summaries
- disappearing feeds
- data that cannot be defended to serious founders or investors
