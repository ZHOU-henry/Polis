# Project Charter

## Mission

Build a disciplined intelligence platform that continuously ingests high-quality
frontier AI signals and produces better judgments about which directions,
products, and markets are gaining real future value.

## Why This Project Exists

AI opportunity selection is currently broken by:

- hype chasing
- weak source discipline
- poor separation between research novelty and commercial viability
- short memory about how technological waves actually unfold

Peras exists to turn that mess into an updateable judgment engine.

## Product Thesis

The first useful version of Peras is not a prediction oracle.

It is a ranking system with:

- source provenance
- explicit uncertainty
- recurring updates
- evidence-backed thesis cards

## Core User Questions

- what is actually moving at the research frontier?
- which directions are moving from papers to tools to products?
- where is the market still underpricing future value?
- which directions are crowded hype and which are durable opportunity?

## Non-Goals

- real-time trading calls
- fake precision forecasts
- social-media trend farming
- one-number scoring without evidence traceability

## First Output Types

- ranked direction maps
- opportunity theses
- momentum and bottleneck notes
- periodic frontier updates
- source-backed change logs
