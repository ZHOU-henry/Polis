# Peras Research Plan 2026

## Objective

Build a credible frontier-signal research base before overbuilding the product
surface.

## Workstreams

### 1. Source Architecture

- refine the source pack
- maintain a clear high-trust vs low-trust split
- identify missing global signal lanes

### 2. Open-Project Landscape

- track open infrastructure for research, benchmarks, code activity, and news
- update the landscape as the ecosystem shifts

### 3. Ranking Method

- define factor weights
- specify uncertainty labels
- test ranking logic against historical cases

### 4. Output System

- expand thesis-card design
- build periodic ranked direction notes
- decide which signals become alerts vs deeper memos

## First Deliverables

- 2026 source pack
- frontier open-project landscape
- current progress synthesis
- open questions registry
- revised research plan and ranking priorities
