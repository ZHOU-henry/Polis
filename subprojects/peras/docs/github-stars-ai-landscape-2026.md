# GitHub Stars AI Landscape 2026

This note tracks high-star AI-related GitHub projects as a lightweight signal
layer for Peras.

## Why This Matters

GitHub stars are not a truth signal by themselves.

But they are still useful for:

- developer attention
- ecosystem gravity
- category visibility
- open-source product pull

Peras should not rank on stars alone, but stars are a meaningful surface inside
the wider signal architecture.

## Interpretation

- Some projects are stars-driven community artifacts and not durable product
  moats.
- Some are infrastructure anchors with real ecosystem gravity.
- Some are useful because they reveal what developers currently want, even if
  the repo itself is not the long-term winner.

## Use Inside Peras

Peras should treat GitHub star data as:

- one open-source attention signal
- one proxy for developer mindshare
- one input to ranking, not the ranking itself
