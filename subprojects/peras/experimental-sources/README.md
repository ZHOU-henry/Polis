# Experimental Sources

This directory is for lower-trust but still potentially useful idea triggers.

## Strict Rule

Everything here is:

- low trust
- brainstorming only
- not directly admissible as evidence
- never enough by itself for ranking, forecasting, or thesis revision

## Mandatory Header

Every note stored here must begin with the following warning logic:

- `TRUST_LEVEL: LOW`
- `USE: BRAINSTORMING ONLY`
- `NOT_FOR: DIRECT EVIDENCE, RANKING, OR THESIS CHANGES`
- `REQUIRES: HIGH-QUALITY CONFIRMATION`

If a note does not contain that warning, treat it as invalid.
