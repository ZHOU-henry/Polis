# Agora - AI Agent Platform

Open-source MVP for an AI Agent platform and operator workspace.

Chinese shorthand: `AI Agent 平台`

## Stage

Stage 1: monorepo scaffold with executable web and API foundations.

## Stack

- `pnpm workspaces`
- `Turborepo`
- `Next.js 16`
- `Fastify`
- `PostgreSQL`
- `Prisma`

## Workspace

- `apps/web/`
- `apps/api/`
- `packages/shared/`
- `infra/`
- `docs/`

## Commands

```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
```
